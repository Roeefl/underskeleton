Superfast to load, with just 15kb minified css, built with mobile in mind.

Designed to be a solid starting point for mobile-first WordPress websites and blogs.

Easy to start changing. Built with SASS and comes with support to NPM, Bower, Gulp and Browser-Sync for a faster development.

About

Underskeleton is a light superfast starter theme with minimalist, dead simple css boilerplate and grid system and just enough styling to get starter with your next awesome WordPress theme.

A full demo site is coming soon. For now you can see the theme test page.

Dev Tools

SASS

Changes get easy with SASS variables and modulated files.

NPM and Bower

All dependencies are managed with NPM and Bower packages.

Is there a new version of Browser-Sync or Skeleton? Just run $ npm update and $ bower update.

Gulp task runner

Comes with Gulp pre-defined tasks for assets management and automatic builds when files get changed.

Browser-Sync

Save time with Browser-Sync automatic page reloads.

Tired of pressing F5 on many devices? Run $ gulp watch-bs and let Browser-Sync do the work.
